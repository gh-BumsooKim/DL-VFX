# DeepLearning VFX

중앙대학교 첨단영상대학원 영상학과 - 딥러닝 시각효과 강의

## Environment
- Ubuntu 20.04 (OS)
- Python 3.7.8
- CUDA 10.1
- cuDNN 7.6
- GTX 1060 3G (GPU)
- Install Package :

```cmd
sudo pip3 install -r requirements.txt
```

### (for python virtual environment user)
- Python3 venv (not Python2 virtualenv, and pyvenv) :

```cmd
cd [dir_path]
python -m venv .venv
source .venv/bin/activate
```
